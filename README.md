Software Requirements
----------------------
* Raspbian “wheezy” for the Raspberry Pi

* raspberry-gpio-python
	https://code.google.com/p/raspberry-gpio-python/downloads/detail?name=python3-rpi.gpio_0.5.2a-1_armhf.deb&can=2&q=

* Tornado Web Server
	http://www.tornadoweb.org/en/stable/

	Download latest version
	wget https://pypi.python.org/packages/source/t/tornado/tornado-3.1.tar.gz
